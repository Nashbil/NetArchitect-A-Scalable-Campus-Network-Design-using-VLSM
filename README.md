# NetArchitect: Scalable Campus Network Design using VLSM

A complete campus network simulation built using **Cisco Packet Tracer**, demonstrating how real-world networks are structured, addressed, and configured.

This project focuses on **efficient subnetting with VLSM**, proper **network segmentation**, and **router/switch configuration** to create a scalable campus infrastructure.

---

## Project Overview

Modern organizations rely on structured network architecture to support multiple departments, devices, and services.

This project simulates a **campus-style network** where multiple networks are connected through routers and switches using structured addressing and routing.

The goal was to design a **clean, scalable, and efficient network topology** that reflects how enterprise networks are planned.

---
